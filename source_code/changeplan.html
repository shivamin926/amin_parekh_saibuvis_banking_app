changeplan

Enter the account holder name and account number. The plan will toggle (SP ⇄ NP). You will recieve a message confirming success or rejection.

This command flips the plan type. Run it again to flip back.

Quick tips

• This command is admin-only.
• The account must exist before the plan can be changed.
• Name must match the account holder for that account number.
• Plan pivots between SP and NP.
