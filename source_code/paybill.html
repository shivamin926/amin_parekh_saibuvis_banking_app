paybill

Standard users can only pay bills from their own account. Company code must be exactly 2 letters (e.g., EC).

You’ll receive a toast message confirming success or explaining why it was rejected.

Quick tips

• Standard paybill cap is 2000 per session.
• Account must be ENABLED and have sufficient funds.
• Company code must be two letters (A–Z).
• Example company codes: EC, TV, HY (demo examples).
