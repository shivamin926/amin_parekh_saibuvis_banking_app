disable

Enter the account holder name and account number to disable the account. You’ll receive a toast confirming success or rejection.

Disabling changes account status to DISABLED.

Quick tips

• This command is admin-only.
• The account must exist before it can be disabled.
• Name must match the account holder for that account number.
• Disabled accounts will be rejected for withdrawal/transfer/paybill/deposit.
